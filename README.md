1.Скачать репозиторий через командную строку git clone или архивом.
2. Импортировать базу данных food.sql
3. В файле app.js  в переменную connection записать поля доступа к базе данных (при необходимости)
4. В папке проекта выполнить команды npm init и npm install
5. Выполнить RUN файла app.js
6. В браузере открыть localhost:3000
